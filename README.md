## Prerequisites

- Install and enable the **Allow CORS** browser extension before loading the HTML file, otherwise the CloudFront `.ts` requests will fail.
  - Chrome/Edge/Firefox: https://mybrowseraddon.com/access-control-allow-origin.html

## Usage

1. Enable the Allow CORS extension.
2. Open `skilluncapped.html` in a modern Chromium-based browser.
3. Paste a SkillCapped video URL, set an optional filename, then choose **Stream** or **Download**.
